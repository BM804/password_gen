# 🔐 Password Generator (Python CLI)

A simple and secure **Command-Line Password Generator** built using Python. It generates strong, random passwords based on user-defined criteria such as length and character types.

---

## 🚀 Features

- 🔢 Custom password length
- 🔠 Uppercase & lowercase support
- 🔣 Includes digits and symbols
- ⚡ Fast and lightweight CLI tool

---

## 🛠️ Tech Stack

- Python
- random module
- string module
